# Dots-Boxes

Dots and Boxes hosted on GitHub Pages.

## GitHub Pages setup

1. Push this repository to GitHub.
2. In the repository settings, open **Pages**.
3. Set the source to **Deploy from a branch**.
4. Select the default branch and **/ (root)** folder.
5. Save. Your site will be available at the URL shown in the Pages panel.

## Local usage

You can open `index.html` directly in a browser, or use a simple static server:

```bash
python -m http.server
```

Then visit `http://localhost:8000`.

## Structure

- `index.html` — entry point with meta tags and links to CSS/JS.
- `styles.css` — base styling placeholder.
- `main.js` — starter script placeholder.
- `favicon.svg` — simple favicon for Pages.
